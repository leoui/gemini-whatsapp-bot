Backup

Create VPS backups, download to Mac, or upload to Google Drive

📦 VPS Backup

Creates a .tar.gz of the bot directory (excluding session and node_modules) on the VPS.

💾 Save Backup to Mac

Download the latest VPS backup directly to a folder on your Mac via SFTP.

☁️ Google Drive Backup

Sign in with Google to upload backups to your Drive. Requires a free Google Cloud OAuth Client ID (one-time setup).

🔧 One-time setup required:
1. Go to console.cloud.google.com
2. Create a project → Enable Google Drive API
3. APIs & Services → Credentials → Create OAuth Client ID
4. Choose Desktop application → Copy Client ID & Secret
✅ Signed in

Enter a folder name (e.g. wabot) or its ID from drive.google.com/drive/folders/THIS_PART

ℹ️ How it works: Takes a VPS backup → downloads it to your Mac via SFTP → uploads to Google Drive. Your VPS credentials never touch Google's servers.

WhatsApp Number

Change the linked WhatsApp account and scan a new QR code

📱 Change WhatsApp Account

This will stop the bot, clear the WhatsApp session on the VPS (auth credentials only — not your bot config), then restart it. You will need to scan a new QR code with the WhatsApp account you want to link.

⚠️ The currently linked WhatsApp account will be disconnected. This is required if you want to switch to a different phone number.

📷 Scan QR Code

After clicking Change WhatsApp Number, the bot will automatically appear here once it generates a QR code. Or click Check for QR to poll manually. Scan it with WhatsApp → Linked Devices → Link a Device.

Import Settings

          
Auto-fill all panels from your live VPS or from a saved XML file. Export your current
            settings for backup.

        

        


          
          

            

              
              
            


            
            

              

                ℹ️ How it works

                Connects to your VPS via SSH and reads the systemd service file (for API keys) and the
                bot's
                config JSON (for persona, contacts, behavior). All fields across all panels are filled automatically —
                nothing is saved until you click Save in each panel.
              

              

                
              

              

            


            
            

              

                ℹ️ Import from saved XML

                Choose a previously exported .xml settings file. All stored values (VPS credentials, API
                keys,
                persona, behavior, contacts, Google Drive creds) will be restored and all panels will be auto-filled.
              

              

                
              

              

            

          


          
          

            

              
✅ Import Summary

              
            

            
Settings imported. Visit each panel to review and save to
              VPS.

            

            

              📝 Next step: All fields are now pre-filled. Visit each panel (Persona, API Keys,
              Contacts, etc.) to review and save your settings. Use Export Settings as XML to create a
              local backup you can restore from anytime — even after reinstalling the app.
            

          


          
          

            
📤 Export Settings

            
Save all your current app settings (VPS credentials, API keys, bot config, Google
              Drive credentials) to an
              XML file on your Mac. Use this to back up your configuration before upgrading or sharing with another Mac.
            

            

              
            

            

            

              🔒 Privacy note: The XML file contains your VPS password and API keys. Keep it in a safe
              location and never share it publicly.
            

          


        

      



      
      

        

          
Uninstall App

          
Completely remove Bot Manager from this Mac

        

        

          

            
⚠️ Uninstall Bot Manager

            
This action will permanently:

            
    
              
  • 🔑 Delete all saved VPS credentials and settings stored on this Mac
    • 
              
  • 🗑 Move the Bot Manager app to Trash
    • 
              
  • 🧹 Clear all app cache and preferences
    • 
            

            

              Your VPS bot is NOT affected. The bot will keep running on your server — only this macOS
              app is removed.
